# Future Location Prediction using Dataiku

## Project Overview

This project predicts the next location (venue category) of a user based on their past movement patterns using a no-code ML platform.

## Tools Used
Dataiku DSS (No-code ML)

## Machine Learning Models:
Random Forest
Logistic Regression
Gradient Boosted Trees

## Features Used
Latitude & Longitude
Hour of the day
Weekday
Previous location (venueId_lag)

## Workflow
Data cleaning and preparation
Feature engineering (time-based features)
Creation of lag features
Model training and evaluation
Performance comparison

## Results
Best model: Random Forest
ROC AUC: ~0.69
Demonstrates strong predictive capability for user movement
